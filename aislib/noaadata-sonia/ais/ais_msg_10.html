AIS Message: utcquery (::)

Description:

Search and rescue position report

Name NumberOfBits ArrayLength Type Units Description
MessageID 6 uint AIS message number. Must be 10
RepeatIndicator 2 uint Indicated how many times a message has been repeated
0: default
3: do not repeat any more
UserID 30 uint Unique ship identification number (MMSI)
Spare1 2 uint Not used. Should be set to zero.
DestID 30 uint Unique ship identification number (MMSI)
Spare2 2 uint Not used. Should be set to zero.
