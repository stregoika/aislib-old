noaadata-py

noaadata-py is a suite of python tools for data used by or produced by NOAA. As of May 2007, the package contains code for marine Automatic Identification System (AIS) messages, extensions for the National AIS (N-AIS) network, and CO-OPS/PORTS waterlevel OpenDAP/SOAP. The software is released under the GNU Public License (GPL) version 2. If needed, other licenses can be negotiated.

XSLT stylesheet outputs for AIS messages are available in ais-messages. If you need a particular message implemented that I have not gotten to, please let me know.
